## Writing Task

### Writing Prompt

**十四、写作。**
**同学们，快来介绍一下你身边的变化吧。**

**Find an old photo and talk about the changes from then to now.**

The template includes the following sections:

* **Title:** Changes in __________
* **Introduction:** I found an old photo of __________. It looks very different now.
* **Comparison Table:** A side-by-side comparison for "In the past..." and "Today..."
* **Conclusion:** I am very happy with the changes. __________ is much better now.



### Model Essay


* **Title:** Changes in **My School**
* **Introduction:** I found an old photo of **my school**. It looks very different now.

| In the past... | Today... |
| --- | --- |
| The school building was small and old. | The school building is big and new. |
| There was only one playground. | We have two big playgrounds. |
| The classrooms didn't have computers or smart boards. | Every classroom has computers and smart boards. |
| We didn't have a library or a gym. | We have a big library with lots of books. There is a new gym where we can play sports. |

* **Conclusion:** I am very happy with the changes. **My school** is much better now.
