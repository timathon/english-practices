### 2. JSON Files Generated
The following files were created in `data/SA1/sa1-u0/`:
- **`sa1-u0-vocab-guide.json`**: Contains all 54 words/expressions with British IPA, parts of speech, meanings, context sentences, page numbers, syllable types, comparison targets, and mnemonic hooks.
- **`sa1-u0-vocab-master.json`**: Features exactly 5 challenges of 10 questions each (50 questions total) using Cloze, Cn2En, and En2Cn types with 6 options and randomized answer indices.
- **`sa1-u0-spelling-hero.json`**: Splits single-word vocabulary items into phonics/syllable chunks with visual/phonetic traps for each chunk.
- **`sa1-u0-sentence-architect.json`**: Comprises exactly 5 challenges of 10 sentences each (50 sentences total) with BrE spelling, noise word distractors, and grammar clues.
- **`sa1-u0-recall-map.json`**: Maps out Welcome Unit themes, vocabulary classes, and basic sentence structure grammar patterns in a tree structure.
- **`sa1-u0-text-navigator.json`**: Groups verbatim reading sentences from "First Impressions" and "Student Profiles" into thematic mindmap nodes with translations, grammar statements, keywords, and highlights.
- **`sa1-u0-writing-map.json`**: Contains basic and advanced model essays for the student profile writing task.
- **`sa1-u0-grammar-wizard.json`**: Tests Basic Sentence Structures (SV, SVO, SP, S VIO DO, SVOC, SVA, SVOA, There be) across exactly 20 multiple-choice questions (2 challenges of 10).
- **`sa1-u0-passage-decoder-s.json`**: Provides sentence-by-sentence reading navigation with 3-option translation multiple-choice questions and vocabulary highlights.
